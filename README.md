# ClaborIA

> IA frugale pour les terrains à faibles ressources.

Laboratoire de recherche & ingénierie personnel. Construire l'IA là où
elle ne fonctionne pas\*\* : peu de compute, peu de données, peu d'argent, terrains
négligés en s'appuyant sur des actifs ouverts existants, prouvé par des PoC
reproductibles et des applications réelles, avec les langues et contextes africains
comme terrain de différenciation.

## Ce que produit ClaborIA

| Pile         | Dossier                         | Description                                                  |
| ------------ | ------------------------------- | ------------------------------------------------------------ |
| Cheat notes  | [`cheat-notes/`](./cheat-notes) | Fiches d'apprentissage publiques, ~1/semaine.                |
| Benchmarks   | [`benchmarks/`](./benchmarks)   | Évaluations rigoureuses reproductibles sur données ouvertes. |
| Apps / démos | [`apps/`](./apps)               | Solutions frugales qui tournent pour de vrai.                |
| Papers       | [`papers/`](./papers)           | Préprints, workshops, rapports techniques.                   |
