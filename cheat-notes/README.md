# Cheat notes

Fiches d'apprentissage publiques. Une notion apprise pour la forge = une fiche.
Objectif de cadence : ~1 par 2 semaines.

## Format

Chaque fiche suit [`../templates/cheat-note.md`](../templates/cheat-note.md) :
problème → notion → exemple minimal reproductible → pièges → sources.

## Nommage

`AAAA-MM-JJ-titre-en-kebab-case.md` (ex. `2026-07-15-onnx-runtime.md`).
