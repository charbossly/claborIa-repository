# Doctrine de ClaborIA

Principes invariants : ils ne changent pas, même quand la techno change.

1. **Reproductibilité avant nouveauté** : chez nous, tout se rejoue en une commande.
2. **La mesure avant le modèle** : mesurer rigoureusement ce que les autres survolent.
3. **L'actif avant le post** : aucune communication non adossée à une preuve.
4. **La composition avant la dispersion** : chaque actif nourrit le suivant.
5. **Apprendre en public** : ce qu'on apprend devient une cheat note ; le savoir composé, visible.

## Ce que ClaborIA ne fait pas

- Collecter des données propriétaires.
- Louer du compute coûteux.
- Publier du contenu déconnecté d'un artefact réel.
